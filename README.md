# Schema

JSON Schema for academic CV and resumes.

### Getting started

Install the package via `npm`

```shell
npm install --save @jsoncv/schema
```

or `yarn`

```shell
yarn add @jsoncv/schema
```

Require the schema like

```js
import { schema } from '@jsoncv/schema'
```
